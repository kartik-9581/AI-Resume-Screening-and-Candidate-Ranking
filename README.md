# 🧠 Resume Screening & Candidate Ranking Web App

An NLP-powered web application that automates resume screening and ranks candidates based on job description relevance.

## 🚀 Overview
This project reduces manual resume shortlisting time by **70%** by automatically analyzing and ranking resumes using Natural Language Processing techniques.

## 🛠️ Tech Stack
- **Python**
- **Streamlit**
- **TF-IDF Vectorization**
- **Cosine & Jaccard Similarity**
- **PDF Text Extraction**

## ⚙️ Features
- Upload single or multiple resumes (PDF)
- Enter Job Description (JD)
- Automatic resume parsing & text extraction
- Similarity-based candidate ranking
- Real-time search & sorted results

## 📊 How It Works
1. Extract text from resumes  
2. Preprocess & clean content  
3. Convert text into TF-IDF vectors  
4. Compute similarity with job description  
5. Rank candidates based on matching score  

## 🎯 Impact
- Automates hiring workflow  
- Saves recruiter time  
- Improves shortlisting accuracy
